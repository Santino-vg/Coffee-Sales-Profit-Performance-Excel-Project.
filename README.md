From Messy Data to a Fully Automated Professional Dashboard! ☕📊

Recently, as a self-improvement project, I took a raw coffee store sales dataset from Kaggle. My goal was to transform this messy data into a clean, dynamic, and visually actionable dashboard.

🎯 Key Project Highlights:

• Data Cleaning & Integration: While analyzing the raw data, I found and removed several duplicate entries. Because the original dataset only included the sales amount per quantity, I successfully integrated "Cost" and "Quantity" metrics from a secondary dataset to make the dashboard truly insightful.
• Full Automation: To clean, format, and organize everything, I used Microsoft Excel and Power Query. The best part of this project is that the entire workflow is automated. Now, whenever new store data is entered, a simple 'Refresh' instantly updates both the dataset and the dashboard.
• Data Visualization: To visualize the insights clearly, I utilized advanced Excel tools, including Pivot Tables, Doughnut Charts, Line Graphs, and Bar Charts.
• Business Value: With a dashboard like this, business owners can effortlessly track their daily sales and profit margins. By simply inputting new data, it becomes much easier to analyze sales trends and make solid, data-driven decisions for the future.
